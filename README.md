# Features
* Auto-parry
    * Logging of animations
    * Builder-styled system (like Linoria)
    * Hard to get working, potential to be good depending on the config
    * Config system for those who cannot bother
* Movement
    * Infinite-jump
    * Speed-hack
    * Jump-power
    * No-clip

**More to come, soon.**
